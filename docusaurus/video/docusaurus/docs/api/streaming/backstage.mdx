## Configuration

<Tabs groupId="examples">
<TabItem value="js" label="JavaScript">

```js
// call level
call.update({
  settings_override: {
    backstage: {
      enabled: true,
    },
  },
});

// or call type level
client.updateCallType('<call type name>', {
  settings: {
    backstage: {
      enabled: true,
    },
  },
});
```

</TabItem>
</Tabs>

## Backstage Permissions

<Tabs groupId="examples">
<TabItem value="js" label="JavaScript">
