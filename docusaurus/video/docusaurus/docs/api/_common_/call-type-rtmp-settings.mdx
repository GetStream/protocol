You can override the default RTMP broadcast layout for a call type with:

<Tabs groupId="examples">
<TabItem value="js" label="JavaScript">

```js
await client.video.updateCallType("livestream", {
    settings: {
        broadcasting: {
            enabled: true,
            rtmp: {
                enabled: true,
                quality: VideoRTMPSettingsRequestQualityEnum._1080P,
                layout: {
                    name: VideoLayoutSettingsRequestNameEnum.SPOTLIGHT,
                }
            }
        }
  },
});
```

</TabItem>
<TabItem value="py" label="Python">

```py
client.video.update_call_type(
    name='livestream',
    settings=CallSettingsRequest(
        broadcasting=BroadcastSettingsRequest(
		        enabled=True,
		        rtmp=RtmpSettingsRequest(
				      enabled=True
			          quality="1080p",
			          layout=LayoutSettingsRequest(
			              name="spotlight",
			          ),
		        )
        )
    ),
)
```

</TabItem>
